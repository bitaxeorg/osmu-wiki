---
title: The BM1370
---

The BM1370 is an undocumented SHA256 mining ASIC from Bitmain. It's mostly used in the Antminer S21 Pro.

## Hashrate

The nominal hashrate of the BM1370 is between 1~1.4TH/s.

|                 |                |
| --------------- | -------------- |
| Price           | New: ~$unknown |
| Efficiency      | 15J/TH         |
| Serial Protocol | UART           |
| Baudrate        | -              |
| Footprint       | -              |
